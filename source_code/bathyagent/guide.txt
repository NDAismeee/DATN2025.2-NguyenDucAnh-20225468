11. MODEL ARCHITECTURE

The model should include the following modules:

11.1 Noise-aware visual encoder
Input:
- RGB image [B, 3, H, W]
- unreliable mask [B, 1, H, W]

Concatenate:
- X_prime = concat(image, unreliable_mask)
- Shape: [B, 4, H, W]

Encoder:
- Start with a simple CNN or U-Net encoder.
- Recommended first implementation: shallow U-Net.
- Output feature map:
  F_vis [B, C, H, W]

11.2 Region-level visual pooling
For each disturbance region R_k:
- Apply masked average pooling over F_vis.
- Output:
  v_k [B, K, C]

11.3 Text projection
Input:
- text_embeddings [B, K, D_text]

Projection:
- Linear(D_text -> C)
- ReLU
- Linear(C -> C)

Output:
- z_t [B, K, C]

11.4 Visual projection
Input:
- v_k [B, K, C]

Projection:
- Linear(C -> C)
- ReLU
- Linear(C -> C)

Output:
- z_v [B, K, C]

11.5 Alignment loss
Use cosine alignment:

L_align = mean(1 - cosine_similarity(z_v, z_t))

Only compute this over valid non-padded regions.

11.6 Expert representation
Aggregate text features:
- Option A: mean pooling over valid text regions.
- Option B: attention pooling.
- Start with mean pooling for simplicity.

Output:
- V_expert [B, C]

Expand to spatial map:
- V_expert_map [B, C, H, W]

11.7 Multimodal fusion
Concatenate:
- F_vis [B, C, H, W]
- V_expert_map [B, C, H, W]

Fusion input:
- [B, 2C, H, W]

Fusion module:
- Conv2d(2C, C, kernel=3, padding=1)
- ReLU
- Conv2d(C, C, kernel=3, padding=1)
- ReLU

Output:
- A [B, C, H, W]

11.8 Depth and uncertainty head
Input:
- A [B, C, H, W]

Output:
- mu [B, 1, H, W]
- log_var [B, 1, H, W]

Convert log_var to variance:
- var = softplus(log_var) + 1e-6

11.9 Soft-gating module
Input:
- fused feature A
- variance var
- unreliable mask M

Concatenate:
- gate_input = concat(A, var, M)
- Shape: [B, C+2, H, W]

Gate network:
- Conv2d(C+2, C, 3, padding=1)
- ReLU
- Conv2d(C, 1, 1)
- Sigmoid

Output:
- alpha [B, 1, H, W]

Final prediction:
- depth_pred = (1 - alpha) * mu + alpha * d_phys

Interpretation:
- alpha close to 0: trust neural prediction.
- alpha close to 1: trust LLM physical prior.


12. LOSS FUNCTIONS

12.1 Heteroscedastic Gaussian negative log-likelihood

For valid depth pixels only:

L_nll = mean_valid(
    ((depth - depth_pred)^2) / (2 * var) + 0.5 * log(var)
)

Use valid_mask to ignore pixels without ground-truth depth.

12.2 Alignment loss

L_align = mean_valid_regions(1 - cosine_similarity(z_v, z_t))

12.3 Optional prior range penalty

This is optional but useful.

If LLM gives d_min and d_max maps:
- Penalize predictions outside the physical range.

L_range = mean(
    ReLU(d_min - depth_pred)^2 + ReLU(depth_pred - d_max)^2
)

12.4 Total loss

Recommended initial version:

L_total = L_nll + lambda_align * L_align

Better full version:

L_total = L_nll + lambda_align * L_align + lambda_range * L_range

Recommended hyperparameters:
- lambda_align = 1e-2
- lambda_range = 1e-3
- learning_rate = 1e-4
- weight_decay = 1e-4
- batch_size = 2 to 8 depending on GPU memory

3. Do not allow LLM prior to dominate everywhere.
